# Atividade 17 - Lógica de Programação (SENAI)

Este repositório contém a resolução de uma lista de exercícios de lógica de programação focada em fundamentos de algoritmos, cálculos matemáticos e estruturas sequenciais.

## 🛠️ Tecnologias e Ferramentas
- **VisuAlg 2.5**: Para o desenvolvimento dos algoritmos em pseudocódigo (Portugol).
- **Draw.io**: Para a elaboração dos fluxogramas.
- **Git/GitHub**: Para controle de versão e hospedagem do código.

## 📋 Lista de Exercícios
Cada exercício possui dois arquivos: um `.alg` (código) e um `.png` (fluxograma).

1. **Soma de dois números**: Algoritmo simples para entrada e saída de dados.
2. **Média de 3 notas**: Cálculo aritmético básico.
3. **Área do Trapézio**: Aplicação da fórmula `A = (B + b) * h / 2`.
4. **Cálculo de IMC**: Cálculo de Índice de Massa Corpórea `IMC = peso / (altura * altura)`.
5. **Conversor de Temperatura**: Conversão de Celsius para Fahrenheit.
6. **Área de Terreno**: Cálculo de área retangular `Área = largura * comprimento`.

## 🚀 Como executar
1. Baixe e instale o VisuAlg.
2. Clone este repositório usando o comando:
   `git clone https://github.com/ThiagoMarcelinoVieira/atv17.git`
3. Abra os arquivos `.alg` dentro do VisuAlg e pressione `F9` para rodar.
